# PostgreSQL 数据库工具 - UI 使用指南

## 界面概览

新的界面采用了类似 VS Code 数据库扩展的设计风格，提供了更专业和高效的用户体验。

### 主要区域

1. **活动栏（Activity Bar）** - 最左侧的图标栏
   - 首页
   - 数据库资源管理器
   - 数据库导出
   - 数据库导入
   - 配置
   - 主题切换（底部）

2. **侧边栏（Sidebar）** - 数据库资源管理器的树形视图
   - 显示所有数据库连接
   - 展开查看数据库、Schema 和表
   - 双击表名打开表数据

3. **主内容区（Content Area）** - 多标签页编辑器
   - SQL 查询编辑器
   - 表数据查看器
   - 支持多个标签页同时打开

## 数据库资源管理器

### 树形结构

```
📁 CONNECTIONS
  └─ 🗄️ database_name
      └─ 📁 Schemas
          └─ 📁 public
              └─ 📁 Tables
                  ├─ 📋 table1 (100)
                  ├─ 📋 table2 (250)
                  └─ 📋 table3 (50)
```

### 操作方式

1. **加载数据库列表**
   - 点击侧边栏顶部的刷新按钮
   - 自动加载所有可用数据库

2. **展开数据库**
   - 点击数据库名称前的箭头
   - 自动加载该数据库的表列表

3. **查看表数据**
   - 双击表名
   - 在新标签页中打开表数据

## SQL 查询编辑器

### 功能特性

- **多标签页支持** - 可以同时打开多个 SQL 查询
- **语法高亮** - 清晰的代码显示
- **执行查询** - 点击"执行"按钮运行 SQL
- **结果显示** - 查询结果以表格形式展示

### 使用方法

1. 点击标签栏右侧的 "+" 按钮创建新查询
2. 输入 SQL 语句
3. 点击"执行"按钮运行查询
4. 查看下方的结果表格

## 表数据查看器

### 功能特性

- **分页浏览** - 支持大数据量表的分页查看
- **CRUD 操作** - 完整的增删改查功能
- **实时刷新** - 随时刷新表数据
- **行内编辑** - 直接编辑和删除记录

### 操作说明

1. **查看数据**
   - 双击左侧树中的表名
   - 数据自动加载到新标签页

2. **新增记录**
   - 点击工具栏的"新增"按钮
   - 在弹出的表单中填写数据
   - 点击"创建"保存

3. **编辑记录**
   - 点击数据行右侧的"编辑"按钮
   - 修改表单中的数据
   - 点击"保存"更新

4. **删除记录**
   - 点击数据行右侧的"删除"按钮
   - 确认删除操作

5. **分页控制**
   - 使用表格底部的分页控件
   - 可选择每页显示 10/20/50/100 条记录

## 主题切换

- 点击活动栏底部的主题图标
- 在亮色和暗色主题之间切换
- 主题设置会自动保存

## 快捷操作

- **双击表名** - 快速打开表数据
- **标签页关闭** - 点击标签页上的 × 关闭
- **多标签切换** - 点击标签页标题切换

## 设计特点

1. **紧凑布局** - 最大化内容显示区域
2. **树形导航** - 清晰的层级结构
3. **多标签页** - 高效的多任务处理
4. **响应式设计** - 适应不同屏幕尺寸
5. **专业外观** - 类似 VS Code 的现代界面

## 技术实现

- **Vue 3** - 现代化的前端框架
- **Naive UI** - 优雅的组件库
- **Tree 组件** - 高性能的树形视图
- **Tabs 组件** - 灵活的标签页管理
- **DataTable 组件** - 强大的表格功能
